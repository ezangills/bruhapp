# BruhApp - ZK Skill Verification Platform

A complete Web3 platform for learning and earning Soulbound Tokens (SBT) using zero-knowledge proofs. Users can generate ZK proofs of their skills and mint non-transferable tokens representing their expertise.

## 🚀 Features

- **ZK Proof Generation**: Generate zero-knowledge proofs for skills
- **Soulbound Token Minting**: Mint non-transferable NFTs representing skills
- **On-chain Verification**: Verify proofs using smart contracts
- **Modern Frontend**: Beautiful React frontend with wallet integration
- **Reputation System**: Build reputation based on verified skills

## 🏗️ Architecture

### Smart Contracts
- **SoulboundToken.sol** - ERC721 token for representing skills (non-transferable)
- **SkillVerifier.sol** - ZK proof verifier contract (generated by snarkjs)
- **ZKSoulboundMinter.sol** - Main contract that combines verification and minting

### Frontend
- **Next.js 15** with TypeScript and Tailwind CSS
- **RainbowKit** for wallet connection
- **Wagmi** for Ethereum interactions
- **Modern UI** with responsive design

## 🛠️ Setup & Installation

### Prerequisites
- Node.js 18+
- npm or yarn
- MetaMask or other Web3 wallet
- Hardhat (for local development)

### 1. Install Dependencies

```bash
# Install backend dependencies
npm install

# Install frontend dependencies
cd frontend
npm install
cd ..
```

### 2. Start Local Blockchain

```bash
npx hardhat node
```

### 3. Deploy Smart Contracts

```bash
# Deploy SoulboundToken
npx hardhat run scripts/deploySoulboundToken.js --network localhost

# Deploy SkillVerifier
npx hardhat run scripts/deploySkillVerifier.js --network localhost

# Deploy ZKSoulboundMinter (update addresses in script first)
npx hardhat run scripts/deployZKMinter.js --network localhost
```

### 4. Update Frontend Contract Addresses

After deployment, update the contract addresses in `frontend/src/config/contracts.ts`:

```typescript
export const CONTRACT_ADDRESSES = {
  hardhat: {
    soulboundToken: '0x...', // Your deployed address
    skillVerifier: '0x...',  // Your deployed address
    zkSoulboundMinter: '0x...', // Your deployed address
  },
  // ...
};
```

### 5. Start Frontend

```bash
cd frontend
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## 🎯 Usage

### 1. Connect Wallet
- Open the application in your browser
- Click "Connect Wallet" and choose your preferred wallet
- Ensure you're connected to the correct network (Hardhat local for development)

### 2. Generate ZK Proof
- Use the ZK circuit tools to generate a proof of your skill
- Save the proof as a JSON file

### 3. Mint SBT Token
- Navigate to "Mint SBT" page
- Enter a skill name (e.g., "Web3 Developer")
- Upload your ZK proof JSON file
- Click "Mint Soulbound Token"

### 4. View Profile
- Go to "Profile" page to see your earned SBT tokens
- View your reputation score and statistics

### Frontend Development

```bash
cd frontend

# Start development server
npm run dev

# Build for production
npm run build

# Run linting
npm run lint
```

### ZK Circuit Development

```bash
# Compile circuit
circom skillProof.circom --r1cs --wasm --sym

# Generate proving key
snarkjs groth16 setup skillProof.r1cs pot12_final.ptau skillProof_0000.zkey

# Generate final proving key
snarkjs zkey contribute skillProof_0000.zkey skillProof_final.zkey

# Export verification key
snarkjs zkey export verificationkey skillProof_final.zkey verification_key.json
```

## 📝 API Reference

### Smart Contract Functions

#### SoulboundToken
- `mint(address to, string skillName)` - Mint a new SBT
- `getSkill(uint256 tokenId)` - Get skill name for token
- `balanceOf(address owner)` - Get token balance

#### ZKSoulboundMinter
- `mintWithProof(address to, string skillName, uint[2] a, uint[2][2] b, uint[2] c, uint[1] input)` - Mint SBT with ZK proof

#### SkillVerifier
- `verifyProof(uint[2] a, uint[2][2] b, uint[2] c, uint[1] input)` - Verify ZK proof
